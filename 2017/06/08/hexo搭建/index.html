hexo搭建

这是我第一次搭建 hexo,根据我自己的实际情况来理解,你也可以参考hexo 官网

配置环境

  1. 搭建node (必须)和 git(必须)

  2. 查看 node 的版本号

  3. 创建一个文件test(使用 sublime 创建,创建后临时放在桌面上)

  4. 切换到 test 目录运行

  5. 安装 HEXO

以上就是所有使用之前的配置

新建 HexoHexo

  1. 将会在指定文件夹中新建所需要的文件(切换到存放 hexo 的文件路径下,初始化 hexo)

  2. 新建后的文件目录

  3. 生成静态页面至public目录

  4. 开启预览访问端口（默认端口4000，’ctrl + c’关闭server）

  5. 访问http://localhost:4000就可以看到本地的博客(必须在执行hexo server后才可以运行本地博客如果关闭就无法访问)

  6. 参考网址
